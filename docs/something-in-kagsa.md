# nlist()
This fucntion is fot create a number list, for example you can use it in for loop
```
write nlist(5)
// [0,1,2,3,4,5]
// from 0 to 5

write nlist(5,zero=false)
// [1,2,3,4,5]
// from 1 to 5
```
